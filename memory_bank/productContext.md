# Product Context: PopFlix

## Проблема пользователя
Пользователи хотят находить интересные фильмы, но сталкиваются с проблемой выбора из огромного количества доступного контента. Им нужна система, которая:
- Запоминает предпочтения пользователя
- Рекомендует фильмы на основе интересов
- Позволяет отслеживать просмотренные фильмы
- Показывает, где смотреть фильм онлайн

## Решение
PopFlix - это веб-приложение, которое:
1. Позволяет зарегистрироваться и настроить предпочтения (жанры, актёры, режиссёры)
2. Получает данные о фильмах из OMDb API
3. Формирует персонализированные рекомендации
4. Позволяет добавлять фильмы в просмотренные, избранное
5. Показывает статистику просмотров
6. Поддерживает тёмную/светлую тему

## Целевая аудитория
- Пользователи, желающие получать персонализированные рекомендации фильмов
- Те, кто хочет отслеживать свои просмотры
- Пользователи, которым важно знать, где смотреть фильм

## Ключевые сценарии использования
1. **Регистрация и настройка** - новый пользователь создаёт аккаунт и указывает предпочтения
2. **Просмотр рекомендаций** - пользователь видит подобранные фильмы на главной странице
3. **Взаимодействие с фильмами** - добавление в просмотренные, избранное, оценка
4. **Поиск** - поиск фильмов по названию, жанру, актёру
5. **Анализ статистики** - просмотр аналитики своих просмотров
