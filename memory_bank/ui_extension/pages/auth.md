# Auth Pages

## /login
**Назначение:** Страница входа пользователя

**Компоненты:**
- LoginForm - форма входа с валидацией
- AuthGuard - wrapper для защиты

**Поток данных:**
1. Пользователь вводит email и пароль
2. Запрос к Supabase Auth
3. При успехе - редирект на dashboard
4. При ошибке - отображение сообщения

## /register
**Назначение:** Страница регистрации пользователя

**Компоненты:**
- RegisterForm - форма регистрации
- AuthGuard - wrapper для защиты

**Поток данных:**
1. Пользователь вводит email и пароль
2. Запрос к Supabase Auth
3. При успехе - редирект на onboarding
4. При ошибке - отображение сообщения

## /onboarding
**Назначение:** Настройка предпочтений пользователя

**Компоненты:**
- OnboardingForm - форма выбора жанров, актёров, режиссёров

**Поток данных:**
1. Пользователь выбирает любимые жанры
2. Добавляет любимых актёров и режиссёров
3. Сохранение в user_preferences
4. Редирект на dashboard

## /dashboard
**Назначение:** Главная страница с рекомендациями

**Компоненты:**
- Header - навигация и поиск
- RecommendationList - список рекомендованных фильмов

**Поток данных:**
1. Загрузка рекомендаций из API
2. Отображение карточек фильмов
3. Возможность взаимодействия с фильмами

## /search
**Назначение:** Поиск фильмов

**Компоненты:**
- SearchBar - поле поиска
- FilterPanel - фильтры
- MovieGrid - результаты поиска

**Поток данных:**
1. Пользователь вводит запрос
2. Запрос к OMDb API
3. Отображение результатов

## /movie/[id]
**Назначение:** Детальная страница фильма

**Компоненты:**
- MovieDetails - полная информация о фильме
- WatchProviders - доступность на платформах
- RatingInput - оценка фильма

**Поток данных:**
1. Загрузка деталей фильма по IMDb ID
2. Отображение информации
3. Возможность добавить в просмотренные/избранное

## /watched
**Назначение:** Список просмотренных фильмов

**Компоненты:**
- WatchedMovieList - список фильмов с оценками

**Поток данных:**
1. Загрузка из watched_movies
2. Отображение с датами и оценками

## /favorites
**Назначение:** Список избранных фильмов

**Компоненты:**
- FavoriteMovieList - список избранных

**Поток данных:**
1. Загрузка из favorite_movies
2. Отображение с постерами

## /stats
**Назначение:** Статистика просмотров

**Компоненты:**
- StatsOverview - общая статистика
- TopActors - топ актёров
- TopDirectors - топ режиссёров

**Поток данных:**
1. Расчёт статистики на основе watched_movies
2. Отображение метрик
